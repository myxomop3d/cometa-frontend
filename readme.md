# cometa-frontend App

This is a new frontend side of cometa application. It's main purpose is to visualize message dataflow thought different applications, brockers, etc..

## Application tech stack

- reactjs
- vite
- tanstack router
- tanstack query
- tanstack table
- shadcn
- react flow

## First dev steps

+ setup project
    + create project structure (file based routing with tanstack)
    + install required dependencies
    + include and configure tanstack router and tanstack query dev tools
+ create simple *Hello, World!* pages [automated-system, flow-graph, components]
+ create sidebar with Links on those pages
